Umami Home Layout
==================

This is a custom layout for the Umami homepage that displays:
- Hero banner with featured recipe
- Featured content cards
- Recipe grid
- Recipe collections footer

To use this layout:
1. Go to Structure > Layouts
2. Edit the "Home" layout (or create a new layout for the front page path)
3. Change the layout to "Umami Home"
4. Save and clear caches
