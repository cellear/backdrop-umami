Image Files for Umami Sample Content
======================================

This directory should contain the image files referenced in data/media_files.json.

To get the images:
1. From a Drupal Umami installation, copy from: web/sites/default/files/
2. From the source repository (if images are included)
3. Use your own food photography

Expected image files (see data/media_files.json for complete list):
- mediterranean-quiche-umami.jpg
- vegan-chocolate-nut-brownies.jpg
- veggie-pasta-bake-umami.jpg
- watercress-soup-umami.jpg
- victoria-sponge-umami.jpg
- gluten-free-pizza-umami.jpg
- thai-green-curry.jpg
- crema-catalana.jpg
- fiery-chili-sauce.jpg
- borscht-with-pork-ribs.jpg
- oatmeal-fruit-syrup-topping.jpg
- dairy-free-milk-chocolate.jpg
- supermarket-savvy-shopping.jpg
- mushrooms.jpg
- carrots.jpg
- baking-mishaps.jpg
- skip-the-spirits.jpg

Note: The sample content module will work without these images,
but they will show as broken image links until added.
